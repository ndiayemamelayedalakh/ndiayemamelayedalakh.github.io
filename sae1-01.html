SAÉ 1-01 : Reporting à partir d’un SGBD

Présentation du projet

Analyse des décès lors de confrontations avec la police aux États-Unis à partir d’une base SQLite.

Démarche

Fonctions Python pour calculer les proportions de victimes armées/non armées, par origine ethnique, et par type d’arme.

Acquis

  • Manipulation de base de données avec Python
  • Utilisation de SQLite3 et Openpyxl
  • Création de graphiques Excel
